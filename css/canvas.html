Canvas

Kesimpulan: Canvas pada HTML memungkinkan pembuatan grafik, animasi, dan visualisasi yang dinamis. Dengan menggunakan JavaScript, kita dapat menggambar berbagai bentuk dan mengembangkan animasi interaktif. Teknologi ini sangat berguna untuk pembuatan game, simulasi, serta data visualization dalam pengembangan web modern.
